# 测试指南：预览器 vs 真机/模拟器

## 🔍 问题说明

您遇到的错误：
```
ErrorCode: 00402015
ErrorDescription: Apply Changes not supported when device type is Previewer.
```

**原因**：预览器（Previewer）不支持某些HarmonyOS高级功能，如应用接续、分布式能力等。

## ✅ 已修复

已对代码进行兼容性处理，现在可以在预览器中正常运行：
- ✅ 添加了 try-catch 错误处理
- ✅ 预览器环境下使用默认断点
- ✅ 不支持的功能会优雅降级

## 🔄 应用接续测试（真机）

### 前置条件
1. 两台HarmonyOS设备
2. 登录同一华为账号
3. 开启Wi-Fi和蓝牙
4. 安装相同应用

### 测试步骤

```
设备A（源设备）:
1. 打开计算器应用
2. 输入表达式，如 "123 + 456"
3. 不要点击等号

设备B（目标设备）:
1. 点击Dock栏的计算器图标
2. 应用自动从设备A迁移过来

验证:
✅ 输入的表达式保持不变
✅ 计算结果一致
✅ 用户操作连续
```

## 🐛 常见问题

### Q1: 预览器显示空白
**A**: 检查控制台错误，可能是组件导入问题

### Q2: 断点不变化
**A**: 预览器不支持窗口尺寸变化，请使用模拟器或真机

### Q3: 应用接续失败
**A**: 
- 确认两设备登录同一账号
- 确认Wi-Fi和蓝牙已开启
- 确认应用已安装

### Q4: 分布式权限错误
**A**: 
- 预览器会忽略权限
- 真机需要用户授权
