## 三、应用接续数据同步

### 在 HomePage 中添加状态同步

**1. 使用 @StorageLink 同步数据**
```typescript
@StorageLink('inputValue') inputValue: string = '';
@StorageLink('calValue') calValue: string = '';
```

**2. 监听数据变化**
```typescript
aboutToAppear() {
  // 确保主题管理器已初始化
  ThemeManager.getInstance();
  this.startCursorBlink();
  
  // 从 AppStorage 恢复接续数据
  let savedInput = AppStorage.get<string>('inputValue');
  let savedCal = AppStorage.get<string>('calValue');
  if (savedInput) {
    this.inputValue = savedInput;
  }
  if (savedCal) {
    this.calValue = savedCal;
  }
}

aboutToDisappear() {
  // 保存数据到 AppStorage
  AppStorage.setOrCreate('inputValue', this.inputValue);
  AppStorage.setOrCreate('calValue', this.calValue);
  
  // 清理定时器
  if (this.cursorTimer !== -1) {
    clearInterval(this.cursorTimer);
    this.cursorTimer = -1;
  }
}
```

## 五、注意事项

1. **数据大小限制**
   - onContinue 传输数据需 < 100KB
   - 大数据使用分布式数据对象或分布式文件系统

2. **生命周期管理**
   - aboutToAppear 初始化
   - aboutToDisappear 清理资源
   - 正确注销事件监听

3. **性能优化**
   - 避免频繁的状态更新
   - 使用 @StorageLink 减少不必要的渲染
   - 合理使用 lazy 加载

4. **兼容性**
   - 检查设备是否支持系统能力
   - 使用 canIUse() 判断 API 可用性
