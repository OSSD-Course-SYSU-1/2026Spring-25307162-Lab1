# 预览器兼容性说明

## ⚠️ 当前配置状态

**预览器兼容模式已启用**

当前配置已禁用以下功能以支持预览器：
- ❌ 应用接续（continuable: false）
- ❌ 分布式数据同步权限

## 🎯 使用场景

### 场景1：预览器测试（当前配置）
```json5
// module.json5
"continuable": false
// 无 requestPermissions
```
✅ 可以在预览器中运行
✅ 响应式布局正常工作
❌ 无法测试流转功能

### 场景2：真机/模拟器测试（完整功能）
```json5
// module.json5
"continuable": true
"requestPermissions": [
  {
    "name": "ohos.permission.DISTRIBUTED_DATASYNC",
    "reason": "$string:distributed_datasync_desc",
    "usedScene": {
      "abilities": ["EntryAbility"],
      "when": "inuse"
    }
  }
]
```
❌ 预览器不支持
✅ 模拟器支持（部分功能）
✅ 真机完全支持

## 🔄 快速切换配置

### 启用完整功能（真机测试）

在 `entry/src/main/module.json5` 中：

1. 修改 continuable：
```json5
"continuable": true
```

2. 添加权限（在 abilities 后面）：
```json5
"requestPermissions": [
  {
    "name": "ohos.permission.DISTRIBUTED_DATASYNC",
    "reason": "$string:distributed_datasync_desc",
    "usedScene": {
      "abilities": ["EntryAbility"],
      "when": "inuse"
    }
  }
]
```

### 禁用分布式功能（预览器测试）

在 `entry/src/main/module.json5` 中：

1. 修改 continuable：
```json5
"continuable": false
```

2. 删除 requestPermissions 配置块

## 🛠️ 自动化脚本（可选）

可以创建两个配置文件：

**module.preview.json5** - 预览器配置
**module.production.json5** - 生产配置

在构建时自动选择正确的配置。

## 💡 提示

- 预览器主要用于UI快速验证
- 响应式布局的核心功能在预览器中仍可测试
- 流转功能必须在真机上测试
- 建议开发时使用预览器配置，发布时切换到完整配置
